Компания «Быстрые деньги» предлагает услуги по выдаче мгновенных займов своим клиентам. Каждое отделение компании имеет своего директора, бухгалтера и несколько кассиров Директор руководит отделением, бухгалтер занимается выдачей и приемом денег, кассиры занимаются оформлением документов и клиентов. Необходимо разработать базу данных для внутренией системы компании.

Любой человек, если ему необходимы деньги, может обратиться для получения займа в любое отделение. Когда клиент приходит в отделение для оформления займа, он должен предоставить свой паспорт, а также указать сумму займа, которую он хочет получить. В каждом отделении ведется статистика по каждому клиенту, по которой клиенты распределяются на несколько категорий. Заём может быть выдан клиенту, или в выдаче может быть отказано. Если клиент пришел впервые в отделение фирмы, то кассир регистрирует нового клиента.

В компании существует несколько отделений для разных категорий пользователей. Каждое отделение может выдать только определенную максимальную сумму, которая может быть недостаточной по запросу клиента. Отделения условно делятся на следующие категории малые (макс. сумма до 100 тыс. рублей), средние (макс. сумма до 500 тыс. рублей), крупные (макс сумма до 1 млн. рублей)

В случае выдачи займа оформляется два документа документ о выдаче клиенту суммы денег и расписка, которая оформляется клиентом. В расписке указывается, какую сумму и в какой срок клиент обязуется вернуть. Каждый клиент может получить займы в разных отделениях фирмы «Быстрые деньги» на общую сумму не более 2 млн рублей. Бухгалтер проверяет и подписывает каждый документ займа.

Существует два основных типа займов, отличающихся по принципу возврата денежных средств. Есть займы с выплатой долга в один момент общей суммой с процентами и займы, возврат средств по которым осуществляется по частям в течение некоторого периода. В документе о выдаче денег клиенту указывается сумма, выданная данным отделением, тип займа по выплатам, основной % за 1 день по займу, штрафной % за 1 день по займу, а также ФИО клиента, его паспортные данные. Для каждого займа вручную или автоматически создается план выплат. Для займа с выплатой общей суммой указывается один момент выплаты в крайний срок с суммой равной сумме займа + % по займу. Для займа с выплатой по частям формируется несколько записей на разные даты. Каждая выплата закрывает равную часть займа и набежавшие проценты. Таким образом, даже при равных частях основных выплат, сумма, набежавшая по% при каждой выплате, уменьшается.

Каждый день бухгалтер проверяет план выплат для всех клиентов. Если клиент просрочил какую-либо выплату, то добавляется запись штрафных выплат со сроком выплат следующий день. Сумма штрафных выплат сумма задолженности штрафной %. Если клиент не платит штрафные выплаты, то каждый день сумма увеличивается по формуле:

сумма штрафных выплат сумма штрафных выплат * (1 + % штрафных выплат + % основной)

Новая запись о штрафных выплатах при этом не создается.

Клиент оплачивает задолженности по плану и присылает платежное поручение. Бухгалтер сохраняет информацию о том, какие выплаты были закрыты каким платежным документом. Платежный документ может закрывать сразу несколько задолженностей, но не может закрывать одну задолженность по частям.

В случае невозвращения денег клиентом на сумму более 1 млн рублей или сроком более 3 месяцев, фирма вправе обратиться к услугам коллекторских агентств. Для этого фирма заключает договор об оказании коллекторских услуг с интересующей их фирмой, отметив в договоре информацию о займе.

Директор может нанимать и увольнять бухгалтеров и кассиров.
